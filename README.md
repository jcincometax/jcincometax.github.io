# JC Income Tax &amp; Office Services — Website

Static, bilingual (English / Spanish) marketing site for **JC Income Tax &
Office Services** in Morgan Hill, CA. Plain HTML / CSS / JS — no build step, no
dependencies.

- **Live site (at launch):** https://www.jcincometaxx.com
- **Preview (staging):** https://jcincometax.github.io
- **Hosting:** GitHub Pages — this repo, `main` branch, served from the root.
- **Status:** 🚧 Staging. The preview is intentionally hidden from search
  engines until launch. See [GO-LIVE.md](GO-LIVE.md).

## Structure

```
index.html  tax.html  dmv.html  notary.html  insurance.html
office-services.html  about.html  reviews.html  contact.html   English pages (root)
404.html                                                        not-found page
es/                                                             Spanish mirror
assets/css/   assets/js/   assets/img/                          styles, scripts, images
sitemap.xml   robots.txt                                        SEO
```

## Editing

Edit the HTML/CSS/JS files directly — there is no build step. The header,
footer, and nav are duplicated per page, so keep them in sync when changed.
